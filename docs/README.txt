# Hub ASSERT (dinâmico • POWER)
- Ícones maskable + monochrome + apple-touch.
- Tema duplo: ASSERT (ciano) / Luxara (preto+dourado).
- Loader no viewer (enquanto app carrega).
- Deep links (#app=KEY) e Resume (#resume=1).
- PWA offline (SW + manifest).

## apps.json
Edite `apps/apps.json` para gerenciar os cards (sem tocar no HTML).
Campos: key, title, desc, icon, url.

## Estrutura sugerida
/Hub_ASSERT_Dynamic_Power/
/PWA_TICO_Ultra/

## Instalar
Sirva com `python -m http.server` e acesse a pasta do Hub. Use o botão “Instalar”.
